# Checkpoints

Thư mục này dùng để mô tả checkpoint mô hình và lưu link tải nếu có.

Không nên commit trực tiếp checkpoint lớn lên GitHub.

Các file lớn như .pt, .pth, .ckpt nên được lưu ở Google Drive, GitHub Release hoặc nền tảng lưu trữ ngoài.

Khi bổ sung checkpoint, cần ghi rõ:

- tên mô hình
- dataset dùng để train
- số epoch hoặc số step
- input format
- output format
- kết quả chính
- link tải
- SHA256 nếu có
